### Optimal System Requirements
- Mac OSX or Linux
- Node.js v5.3.0
- PhantomJS must be installe on local/server systems
- Further testing needed to source our requirements

### Rapid Start Toolkit including these features:

- React.js
- Node.js
- Express.js
- Babel.js 6.3.x ES7 Stage 0
- GraphQL
- Relay ready
- Sequelize SQL ORM
- sqlite3 database
- PhantomJS SEO
- Passport.js with oAuth2 support
  - SQL User database
  - Github
  - Facebook
  - Twitter
  - Google Plus
- Deployment with the following suites:
  - PM2
  - Microsoft Azure
  - Heroku
